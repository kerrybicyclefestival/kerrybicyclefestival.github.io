Frame X

FEATURING: Fiona Reilly// Grace O’Sullivan // Catherine Green // Dymaxicorp // Shane Crotty // Fiona O’Dwyer // Garry Walsh //Michael Cotter // Studioseventyseven // Chris Hill-Scott // Pete4130 // Martin Gibbins // Chris Judge // Mark Wickham // Luke Prendeville // Noel Molloy // Aoife Hand // Hector Castells // Fionan O’Connell // Aurelie Montfrond // Keith o’Faoláin // Barry McGoo

Due to the success of last years exhibition Frame the bicycle themed art exhibition will be returning this year, to the Siamsa Tíre gallery.
The exhibition will be launched at 6pm on Thursday 13th of May and will run in the  for the duration of the festival.

We would love you to join us at the opening for some drinks and nibbles

You can check out some photos from last years exhibition on Siamsa Galleries blog

Venue: Siamsa Tire Gallery
Dates: Thursday 13th May [Launch]
Time: 6pm
